TO RUN THE PROGRAM:
1. Navigate inside the folder containing the Tatai Beta.jar file using terminal
2. Add executable permissions to the jar (using the command: chmod +x ./Tatai\ Beta.jar)
3. Run the jar from the terminal (using the command: java -jar ./Tatai\ Beta.jar)
